TEMA DEL PROYECTO
==================
Lanastina - Tienda de patrones de crochet

DESCRIPCION 
===========
Sitio web con tienda de patrones de crochet, incluye:
- Home con catalogo de productos
- Blog de noticias/tutoriales
- Carrito de compras (visual)
- Base de datos MySQL con 3 tablas (users, patterns, blog_posts)
- Bootstrap para estilos
- Modo oscuro/claro 

INSTALACION
===========

1. Clonar el repositorio de GitHub https://github.com/valucrak23/Lanastina.git

2. Iniciar Laragon

3. Instalar dependencias de Composer (desde terminal en la carpeta del proyecto):
   composer install

4. Crear base de datos MySQL llamada: lanastina

5. Crear archivo .env en la raíz con esta configuración:
   
   APP_NAME=Lanastina
   APP_ENV=local
   APP_KEY=
   APP_DEBUG=true
   APP_URL=http://localhost
   
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=lanastina
   DB_USERNAME=root
   DB_PASSWORD=

6. Ejecutar en terminal de Laragon:
   php artisan key:generate
   php artisan migrate
   php artisan db:seed
   
   Esto creará las tablas y cargará datos de ejemplo:
   - users (1 usuario administrador)
   - patterns (6 patrones de crochet)
   - blog_posts (5 artículos del blog)

7. Acceder en navegador:
   http://localhost/Lanastina/public
   
   Rutas:
   - / (home) → Tienda de patrones
   - /blog → Blog/Noticias
   - /carrito → Carrito de compras

NOTA: La carpeta vendor/ no está incluida en el repositorio (gitignore).
Se genera automáticamente al ejecutar composer install.
